# Project Nname : Finesse 

#### Overview : Help clientele manage their assets by estimating their earnings and expenditures for the previous and next year, regulating debit payments and saving for retirement.

#### As a cliente I should be able to do the following :
- Fill the need informations.
- Choose the wanted service:
   1- To Calculat all Last Year Spending.
   2- To Estimate Next Year Spending.
   3- To Get Information On Your Debit.
   4- To Calculat How Many Years Left For Your Retirement.
   5- To Calculat The Needed Raise In Your Salary.
   6- To Exit The Programme.

#### Usage : Explain to the client how to use the project. 
 - Type in your info.
 - Choose the wanted service.
 - View the result for wanted service.
